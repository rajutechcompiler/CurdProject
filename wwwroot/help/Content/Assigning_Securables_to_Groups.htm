Assigning Securables to Groups

  1. Click Admin > Security.
  2. On the Permission tab, select the securable Type with which you want to work.
  3. Select one of the Groups, then one of the Securables.
  4. In the Permission area of the screen, set the appropriate permissions for members of the selected group to the selected securable.
    To quickly copy permissions from one group to another:
    1. Select the group permissions that you want to copy.
    2. Click Copy.
    3. Press CTRL and select the group and securables combination to which you want to copy the permissions.
    4. Click Paste.
  5. Click Save.
