Triggering a Workflow

If a workflow has been configured, you will see a button for it to the left of the Attachments button in the data grid. Any workflows are also listed in the Tools > Workflow menu.

The workflow is tied to a Linkscript, and will prompt you to enter data in one or more screens. For example, a Status button may be configured to allow you to update a recordâ€™s status.
