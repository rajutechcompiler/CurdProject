Applying Retention Codes Manually

If necessary, you can manually apply a disposition code to an entire table.

To view a list of available disposition codes, click Reports > Retention Reports > Retention Codes. This list will help you identify the appropriate code to apply.

To manually apply a disposition code:

  1. Click Retention > Retention Code Maintenance > Reassign Retention Code.
  2. Select the record to which you want to apply the retention code.
  3. Choose the existing (”old”) retention code and a new retention code, then click Ok.
