Adding Records

Open the table to which you want to add a record, click New, enter the necessary data, then click Save.

Note that:

  • Some fields may have default values.
  • Some fields present a checkbox: you simply check or clear the box.
  • Some fields may offer specific controls, for example, clicking in a date field shows a calendar control.
  • Some fields present drop-down lists of possible values.

To add a child record, first select the parent record in the parent table, then click the link to the child table and create a new record as described above.
