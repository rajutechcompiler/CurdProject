Working with Requests

Fulfilling Requests

To delete a request: in the tracking details (see Checking Current Location, Wait List and Tracking History of Record) click Delete Request adjacent to the request.

  1. Click Reports > Requestor Reports > New Requests.
  2. Select the requests you plan to fulfill.
  3. Click Send Checked Items to Pull List. This changes the status of all outstanding requests to "in process".
  4. Use the pull list to locate and deliver the requested records. Transfer the record using the barcode tracking module or the on screen transfer button.

Note:  Records can be transferred only to active employees.

Manually Marking Requests as Fulfilled

  1. Open a pull list.
  2. Right-click on the request and choose Details.
  3. Check the Fulfilled checkbox.

Requesting Exceptions

File requests that cannot be filled are often flagged as Exceptions. Such requests will appear on the Exceptions report to be managed by a supervisor or to ensure special attention is given to the requested item.
