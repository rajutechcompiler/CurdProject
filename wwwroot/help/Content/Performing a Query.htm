Performing a Query

Each time you open a view, by default you are prompted to specify which information you want to see (for example, a specific customer number, document type, due date, customers with an address in a specific province/state, etc.). This is called a "query".

Note:  To disable this behavior, in the Query page clear the Always show basic query when opening a view option. By default, the last query is displayed. To prevent this behavior, clear the Remember this query checkbox.

Performing a Basic Query

Enter values in one or more of the available fields, or leave all fields blank to retrieve all records, and click OK. Multiple field entries are treated as “AND”; that is, only records that meet all criteria will be displayed.

To re-display the query screen to modify selections, click Query. To clear the query screen of any residual filter values, click , and then click Clear.

Advanced Query

  1. Open the view within which you want to search.
  2. On the Query page, click Advanced. (If a view is already open, click Query to display the Query page.)
  3. Select the Column you want to search, then enter a value in the Filter field. Enter any additional values in the Or fields.
  4. Repeat as appropriate (new rows appear as you work). Multiple rows are treated as “AND”; that is, only results that match all entries will be displayed.
  5. Click Apply. The list of entries changes to show just those matching the specified values.

Note: If the query did not work correctly or did not work as you expected, click Show Where Clause to display the syntax of the search so you can make your corrections. For example you might have chosen a numeric field but entered text values.
