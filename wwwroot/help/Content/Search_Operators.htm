Available Search Operators

=

Finds all entries with fields that are exactly equal to the specified values.

>

Finds all entries with fields that have a value greater than the specified value.

<

Finds all entries with fields that have a value less than the specified value.

>=

Finds all entries with fields that have a value greater than or equal to the specified value.

<=

Finds all entries with fields that have a value less than or equal to the specified value.

<>

Finds all entries with fields that have a value that is not equal to the defined value.

IN

Finds all entries with fields that contain the search word.

BEG

Finds all entries with fields that begin with the search character (letter or number).
