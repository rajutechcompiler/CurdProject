Opening the Attachment Viewer

To view or otherwise work with attachments, use the Attachment Viewer. While the viewer is open, selecting a record displays the attachments associated it.

Note:  There are currently two versions of the Attachment Viewer: one runs on Internet Explorer (9.0-11+, but not Edge), and the other runs on Chrome and Safari (not Firefox). These versions have different functionality which is noted in this documentation.

To open the Attachment Viewer, from an open table, click (if the icon has a “+” sign, the record already has one or more attachments; clicking the icon will still open the Attachment Viewer).

The Attachment Viewer opens in a new tab. You can have multiple viewers open: each operates independently.

The Attachments Viewer shows thumbnail versions of an attachment and its individual pages on the left, and a larger view of the selected attachment/page on the right. To change the default view (change the size of the thumbnail, control which panels are shown) use the View menu.

Note:  The Attachment Viewer may be configured to display all attachments for the selected record as well as any attachments for any child records associated with the selected record; click . Child attachments are indicated with a light grey border around the thumbnail in the Attachment Viewer, and can be viewed and managed the same as those attached directly to the selected record.
