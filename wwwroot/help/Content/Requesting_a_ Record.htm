Requesting a Record

  1. Locate and select the record(s) you want to request.
  2. Click and select Request or, on the Tracking Viewer, click Request
  3. Select the requestor for whom the request is being made.
  4. By default, the request is for the next calendar day, and of standard priority. If necessary, change these values.
  5. Optionally, add Instructions for the person who will be fulfilling the request. For example, if requesting a box you might want to have it confirmed that the box contains the specific file you are seeking.
  6. By default, the request is added to the wait list if the record has already been checked-out. To cancel the request, clear the Place already checked out items on to wait list option.
  7. Click Request.

Note:  You may receive an email notification when your requested records are available.
