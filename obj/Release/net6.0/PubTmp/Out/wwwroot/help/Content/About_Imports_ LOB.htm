About Imports

You may want to have data imported into TAB FusionRMS for a variety of purposes, including adding records to the database or updating tracking information for existing records. You need to set up the imports that users will run as required. Imports define the location of the file, where the data is to be imported into, and other elements. You can import data from .csv or Excel files. This definition is stored in an "import load", which can be run on its own or as part of an import job.
