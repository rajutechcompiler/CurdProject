Working with Attachments

The following procedures are done from the Attachment Viewer.

Printing or Emailing Attachments (Internet Explorer only)

To print or email one or more pages of an attachment, click File > Print or File > Email and select Current Page/Attachment or All Pages/Attachments.

If you need to print or email multiple attachments from different records or of different types, use the Attachment Cart. The Attachment Cart is independent of the attachments displayed in the Attachment Viewer, and is used to assemble multiple attachments, of different documents or from different areas, into a collection (batch) that can be exported to a specific folder, emailed or printed. You may choose to use the cart to simply consolidate documents for yourself, or to provide them to someone who does not have access to TAB FusionRMS.

To view the Attachment Cart: click in the Attachment Viewer or click View > Panes > Attachment Cart.

To add attachments to the Attachment Cart, drag the attachment thumbnail to the Attachment Cart pane (or right-click on it and select Add to Cart). If necessary, select a different record to display its attachments and add additional attachments to the Attachment Cart. When you are ready, click Email, Export, or Print as needed. If you chose Email, a PDF is created for each attachment.

Attachments remain in the Attachment Cart for the current session, but you can click Clear to manually empty the cart.

Adding Pages to Attachments (Internet Explorer only)

You can add a page to a document of the same type. Pages can only be added, deleted, or rearranged on a checked out attachment. When checked back in, it is saved as a new version.

  1. Select the attachment to which you want to add a page, then click File > Check Out.
  2. For Word documents, right-click on the thumbnail and click Open in Native Application. Make the changes in Word and save the document.
    For other attachment types, view the Pages thumbnail pane, then click File > Add Page(s) or drag-and-drop the new image to the thumbnails panel.
  3. Click File > Check In.

Opening an Attachment in its Native Application (Internet Explorer only)

Depending on the file type, you may be able to open it in its native application (e.g. Word, Excel). An icon in the thumbnail pane will identify the major file types for non-image files.

Right-click on the thumbnail and click Open in Native Application. The file opens in a read-only temporary file unless you checked it out first. If you check out the attachment and then open it in its native application, you can edit the attachment fully and then check it back in again (it will be saved as a new version).

Deleting an Attachment

Internet Explorer:

Click File > Delete > Entire Attachment.

To delete a page from an attachment, you must first check out the attachment: click File > Check Out, then select the page to be deleted and click File > Delete > Current Page.

Working with Attachment Images

Internet Explorer:

To manage the visibility of the Attachment Viewer or attachments themselves, use the controls below the thumbnail panel or on the Tools menu of the Attachment Viewer.

You can:

  • change the area shown in the right panel (Drag)
  • zoom in on area selected in the right panel (Zoom)
  • display a magnifier that can be used to magnify an area (Magnify)
  • copy a selected area in the right panel to the clipboard (Copy)
  • rotate the image (Tools > Rotate/Save Right, Rotate/Save Left)

A more expanded set of controls is available if you right-click on the thumbnail pane, allowing you to:

  • select all thumbnails (not all functions are available when multiple attachments or pages are selected)
  • change the sort order for thumbnails
  • cut/copy/paste/delete/rename page

Right-clicking on the preview allows you to:

  • navigate to the previous/next attachment for the record
  • navigate through the pages of the current attachment
  • zoom in/out on the page
  • print the page
  • email the current page, current attachment or all attachments for the record
  • rotate the page (Note: For file types other than CSV, the rotation is saved; otherwise the rotation is temporary for viewing purposes only.)
  • view detailed information about the attachment
  • access the same controls as at the bottom of the thumbnail view
  • add notes to the attachment

Other Browsers:

  • View tab: rotate, magnify, fit to width/height of window, show or hide thumbnails.
  • Page tab (or in the toolbar below the attachment): navigate through pages of an attachment, “get text” that can be copied (see below), show multiple pages per screen, and display pages vertically/horizontally.
  • Interactive tab: pan, zoom, zoom to, magnify, or use Inertia Scroll (which keeps scrolling even after you let go of your mouse)
  • Preferences tab: highlight links

Copying Attachment Contents to the Clipboard (Internet Explorer only)

To copy a section of an attachment to the clipboard as an image, click Tools > Copy, then select the portion to be copied. If you want to copy the entire page to the clipboard, click . Regardless of the attachment type, the content is copied as an image.

To convert a selected section of an image to text using OCR, click Tools > OCR to Clipboard and select the portion of the attachment to be processed. The converted text is placed on the clipboard. Note that accuracy of the OCR results may be affected by the quality of the source image; you should review the results and correct any errors.

Recording Notes About an Attachment (Internet Explorer only)

To record a note about the entire attachment that is visible from any page, e.g. “official record pending approval”, click and enter the note text. Your user name and the date are appended to the note. Multiple notes can be recorded for an attachment.

To record comments directly on an individual page, use an annotation note or text. These are only visible when that page is viewed.
