Producing Labels

You can use TAB FusionRMS to produce labels for objects that are to be tracked. You can also produce replacement labels when necessary. Note that your organization may use different label formats: you will need to know which format you want to produce. Each table view may be configured to display an indicator of if or when a label was printed for a record (e.g. a Printed check box or date).

  1. Load the printer with the appropriate label stock.
  2. Open the view containing the records for which you want to produce labels, and select the records
    (or click the prompt in the Task List, if appropriate).
  3. Click and choose the desired label type (TABQUIK or Black & White).
  4. Select the appropriate Design and label Stock.
  5. If you are printing on a partially-used sheet of labels, select which label position to Start Printing with.
    Tip: Select Draw Label Outlines and Preview Data to help guide you to which position to choose.
  6. Click Print.
