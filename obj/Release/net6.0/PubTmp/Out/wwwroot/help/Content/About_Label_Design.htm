About Label Design

A label design acts as a template: it determines what type of information will appear on the labels, and where the information will be placed on the label. You will need to create at least one label design for each table from which labels are to be printed.

Label designs must be associated with a view in order to be available to users.

Label designs are made up of several basic types of objects:

  • text objects, which present static text on a label (e.g. “confidential”)
  • barcode objects, which are associated with the unique ID field of a table resulting in a unique bar code for each record label printed
  • field objects, which present the contents of a record field, resulting in field-specific text for each label printed .
