Running Reports

To run a report:

  1. Click on the Reports workgroup, click the report category, then click on the report name.
  2. The report output is shown on the screen. If necessary, use the controls below the report to set how many rows of data will be shown per report page, and to navigate through the report pages. Some reports (such as Pull Lists) allow you to select criteria, which dynamically changes the report output.
  3. You can Print or Close the report.

Note:  You cannot export report data. If necessary, you can export data from a table (see Exporting Data).
