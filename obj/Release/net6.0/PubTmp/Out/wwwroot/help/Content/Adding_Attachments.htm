Adding Attachments

There are several ways to add an attachment:

  • (Internet Explorer) Drag-and-drop onto the thumbnail pane of the Attachment Viewer - you will be asked if you want to rename the attachment.
  • (Internet Explorer) Via the File menu in the Attachment Viewer - you will be asked if you want to rename the attachment.
  • (Other Browsers) Click on a record that does not yet have any attachments, or in the Attachments menu in the Attachment Viewer.
  • Use the Add-Ins menu in Microsoft Word, Excel or Outlook (version 2003 or higher, 32bit) to send the document to TAB FusionRMS - you will be prompted to select the Linkscript to be used to index the file. Additional prompts may appear depending on the configuration of the Linkscript.
