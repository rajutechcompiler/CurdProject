About Security in TAB FusionRMS

Security is controlled through group membership: users are not assigned access individually, but are granted the access rights of the group(s) to which they are assigned.

Access to each type of information (attachments, reports, etc.) and each securable element (report, table, etc.) can be controlled, as well as the type of action that can be taken on that element. Access is controlled at two levels and on two separate tabs: system-wide access (Securables tab) group-level access (Permissions tab). Securable-based security applies broadly: for example, if the ability to print the Tracking report is not selected on the Securables tab, no group can be granted the permission to print that report. As a result, it is highly recommended that securable-level permissions be defined before group permissions.
