About the User Interface

The horizontal bar across the top of the user interface offers access to specific areas of functionality: clicking on a function name (such as Label Manager of Tracking) opens that function in a new tab. The exception to this is Tools, which offers a menu of tools, and About, which simply shows the product version.

Most functions present, in the vertical navigation panel, a set of expanding/collapsing sections. To hide or show this panel, click anywhere in the vertical bar on the right of the navigation panel (navigation bar). Note that if the contents of a section cannot all be displayed at once, a grey bar is shown to the left of the dividing bar (scroll bar): use it to scroll through the navigation panel contents.

In the bottom left corner of the user interface you see the name of the server to which you are connected, the database with which you are working and your login name.
