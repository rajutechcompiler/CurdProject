About Imports

Your administrator will instruct you if itâ€™s necessary to import data into TAB FusionRMS. Imports are typically used to make changes to existing data or to load data from an external source.

Running an Import

  1. Click Import. The Import Wizard opens.
  2. Select the appropriate import from the Saved Imports list. An import that is identified as [job] contains multiple import load files that will be run together.
  3. To have the import process continue even if errors are encountered, check Quiet Processing. If you do not check this option, the import process will stop if it encounters an error. If you check this option, errors are reported at the end of processing. Use the import report to view any errors.
  4. Click Run. The import runs in the background.
