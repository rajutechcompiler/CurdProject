Working with Versions

When a file is attached to a record, it cannot change. If you want to do anything to it, e.g. manage pages or add annotations, you must create another version. Multiple versions of the same attachment can be associated with a record.

To view a different version of an attachment, click View > All Versions for Attachment <n> (Internet Explorer) or select from the Versions list above the attachment (Other Browsers). If you do not have permission to view all versions, you will only see the most current version.

Adding a New Version of an Attachment

Internet Explorer:

To manage pages or annotations, check out the attachment (File > Check Out). Make your changes and then check it back in - it is saved as a new version. Alternatively, create a new version in one of the following ways:

  • Display the Versions thumbnails in the Attachment Viewer, then click File > Add Version(s) and retrieve the new file.
  • When you add a new attachment and one already exists, you may be prompted to add it as a new attachment, or as a new version or new page to an existing attachment.

Other Browsers:

On the Attachments page, click and retrieve the new file.

Deleting a Version

Internet Explorer: A version must be checked out before it can be deleted.

Click File > Check Out, then click File > Delete > Entire Version.

About Official Records

An “official record” is the authoritative version of an attachment. In Internet Explorer, this is identified by a certificate icon in the lower left corner of the attachment thumbnail as well as the Attachment Information pane.

In other browsers, this is indicated by an icon on the page thumbnail.

If the Official Record feature is supported for a table, the first version of an attachment added to a row will be considered the official record. If a new version is added or an image has been redacted or annotated, the new or latest version can be designated as the new Official Record.

Official Records can be modified. An Official Record Reconciliation report is available that shows changes made to attachments marked as official. The printed report includes a cover sheet with a signature box and a printed version of the official record.

Internet Explorer:

To mark a different version of an attachment as the “Official Record”, either:

  • if you need to make any changes: check out the original, modify it, and then click File > Check In as Official Record. When you then click View > All versions for Attachment <n>, you will see that the newest version is now marked as Official.
  • if you don’t need to make any changes: select the version and click View > Mark as Official Record.

Other Browsers:

To mark a different version of an attachment as the “Official Record”, check it out, modify it, and then click to check it in as the official record.
