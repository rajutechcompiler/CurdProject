Managing Views

To add a view:

  1. In the Admin Manager, in the Views workgroup, click on the table for which you want to add a view: the current views for that table are shown.
  2. Click Add.
  3. Enter a Name for the view.
  4. Optionally, check Searchable View.
  5. Optionally, set the Max records (maximum number of records) the view can show.
  6. Enter the SQL statement that will populate the view, optionally clicking Move filters into SQL statement.
  7. Choose whether the view will be visible in the Task List and, if so, the name that will appear in the task list for the view.
  8. Add the columns to be included in the view, then click Apply.

To change the order of views:

  1. In the Admin Manager, in the Views workgroup for which you want to change the view order: the current views for that workgroup are shown.
  2. Use the arrow buttons to change the order of the views.
